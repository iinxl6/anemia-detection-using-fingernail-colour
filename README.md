# anemia-detection-using-fingernail-colour
Hybrid pipeline for anemia detection from fingernail images: unsupervised feature learning &amp; pseudo-labeling (Autoencoder, PCA, K-means++) followed by supervised classification (SVM, Random Forest, ResNet50)
